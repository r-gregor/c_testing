# Prefix Tree (Trie) Implementation in C

## Quick Start

### Build

```console
$ ./build.sh
```

### Dump the Trie as SVG (Requires [Graphviz](https://graphviz.org/))

```
$ ./trie dot
$ xdg-open trie.dot.svg
```

### Autocomplete prefix

```console
$ ./trie complete Ap
```
